This is my personal fullstack MERN portfolio where I present my work, technical skills, and projects in an interactive way. It combines both frontend design and backend functionality to create a complete user experience.

The portfolio highlights my journey from building responsive UIs to working with backend systems and databases.

##  Installation & Setup  

### 1. Clone the repository  
```bash
git clone https://github.com/anisha00207/MERN_PORTFOLIO.git
cd MERN_PORTFOLIO
```

### 2. Install dependencies  

#### Backend  
```bash
cd backend
npm install
```

#### Frontend  
```bash
cd frontend
cd client
npm install
```

### 3. Environment Variables  

Create a `.env` file in the server folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

### 4. Run the project  

#### Start backend  
```bash
npm start
```

#### Start frontend  
```bash
npm run dev
```
